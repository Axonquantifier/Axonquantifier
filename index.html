Tutorial - Example of how to quantify axons from folders of .tiff/.jpg/.png files.

First download R code/Sample data [here]https://github.com/Axonquantifier/Axonquantifier/tree/gh-pages) You will need the sample data or your own images, as well as the Axonquantifier.R file Note, you will need to modify this code to point towards your code/files

  ### To install required packages, uncomment and run this
  # source("http://www.bioconductor.org/biocLite.R")
  # biocLite("EBImage")
  library("EBImage")
  source(".../.../R/Axonquantifier.R")

  ###To obtain mean quantification for folder of images
  Axonquantifier_data<-Image.process.Individual(".../.../Treatment_Folder", ".../.../Output_Directory","Colour")

  ###To obtain image level quantification for folder of images
  Axonquantifier_data<-Image.process.Average(".../.../Treatment_Folder", ".../.../Output_Directory","Colour")
